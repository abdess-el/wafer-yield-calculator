# Wafer Yield Calculator

A simple web tool to calculate semiconductor manufacturing yield using Murphy's model.

## Live Demo

[(https://abdess-el.github.io/wafer-yield-calculator/)/]

## Formula

Yield = (1 - e^(-D × A)) / (D × A)

- D = Defect Density (defects/cm²)
- A = Die Area (cm²)

## How to use

1. Adjust Defect Density slider
2. Adjust Die Area slider
3. View yield percentage instantly
